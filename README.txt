DhelmScreener is a desktop application as well as a javafx library which is both a stock screener and as well as trading terminal.
The stock screener is implemented using our Dhelm  project libraries.
The purpose behind  DhelmScreener is to bring automation to your stock picking strategy in an affordable way.
It is built on the top of Kite developer APIs, provided by Zerodha.
To visualize the working principal of DhelmScreener, this DhelmScreener 1.0.0-Demo,  is being provided.

To run the software we are providing no data. 
You have to feed your own data using your own APIs.

To explore the usage, navigate through the wiki.

To download the appropriate version, visit the release section.

If you have any query you can raise an issue.
